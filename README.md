A modern, responsive, and performance-optimized personal portfolio website designed to showcase projects, skills, education, and achievements. Inspired by Apple's minimalist design language, it features smooth scrolling, responsive navigation, custom modals, stats count-up animations, and a functional contact form integration.

---

## 🚀 Key Features

* **Apple-Inspired Minimalist UI:** Features custom glassmorphism components, premium typography (Inter/Poppins), elegant micro-animations, and CSS transitions.
* **Technical Toolkit Showcase:** A categorized grid displaying proficiency across programming, web development, AI, cybersecurity, databases, and environments.
* **Featured Projects Showcase:** Highlighted project cards (such as EcoLens AI, CyberGuard AI, and SomaSense AI) complete with technical badges and **"Learn More"** interactive overlay modals.
* **Animated Statistics Dashboard:** An interactive metrics section with dynamic counter-up animations when the user scrolls into view.
* **Responsive Layout:** A mobile-first design with a custom mobile drawer navigation menu and a floating scroll-to-top button.
* **Interactive Contact Form:** A beautifully styled user intake form with full input validation and animated loading indicators.

---

## 🛠️ Technical Toolkit

* **Core Structure & Logic:** HTML5, CSS3 (Vanilla), and ES6+ JavaScript.
* **Design Systems & Fonts:** Google Fonts (Inter, Poppins), Font Awesome Icons.
* **Animations:** Pure CSS keyframes for load-in animations and intersection observer triggers for on-scroll reveals.

## 💻 Local Development

Follow these steps to preview and run the project on your local machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Itssonia7/portfolio-deployed.git
   cd portfolio-deployed
   ```

2. **Run a local development server:**
   * Using **Python (Recommended)**:
     ```bash
     python3 -m http.server 8080
     ```
   * Using **NodeJS (serve)**:
     ```bash
     npx serve .
     ```

3. **View the website:**
   Open [http://localhost:8080](http://localhost:8080) (or the port specified by your tool) in your web browser.

## 🌐 Deploy to Netlify

You can quickly deploy this portfolio using one of the following methods:

### Method A: Drag & Drop (Netlify Drop)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the root repository folder onto the page.
3. Your site will be live instantly!

### Method B: Git Integration (Continuous Deployment)
1. Log in to your [Netlify Dashboard](https://app.netlify.com/).
2. Select **Add new site** > **Import an existing project** > **GitHub**.
3. Connect your repository (`Itssonia7/portfolio-deployed`).
4. Set the **Build Command** to *empty* and **Publish Directory** to `.`.
5. Click **Deploy Site**. Every push to the repository will automatically trigger a new deployment.
